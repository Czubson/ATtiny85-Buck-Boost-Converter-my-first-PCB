# ATtiny85-Buck-Boost-Converter-my-first-PCB
This is a simple yet functional DC/DC Buck-Boost converter controlled by an ATtiny85 microcontroller. The design allows for both step-down (buck) and step-up (boost) conversion, with output using a feedback signal.

Inverting Buck/Boost converter Schematic
<img width="1354" height="675" alt="image" src="https://github.com/user-attachments/assets/e7d468c2-b755-427e-9a59-094943d03bed" />

Components:
- 1 x ATtiny85-20P
- 1 x MCP602-I_P OPamp
- 1 x L7805 voultage regulator
- 1 x IRLZ44N MOSFET-N
- 3 x 1N5819 Schoottky Diode
- 6 x 10k立 THT resitors
- 2 x 4.7k立 THT resitors
- 1 x 47立 tht resistor
- 1 x 100k立 Potentiometer
- 1 x 33uH Inductor
- 2 x 100nF THT Capacitors
- 1 x 220uF Electrolytic capacitor
- 1 x 100uF Electrolytic capacitor
- 2 x Screw terminals
- 1 x Jack-DC Input
