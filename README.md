# ATtiny85-Buck-Boost-Converter-my-first-PCB
To jest prosty, ale funkcjonalny DC/DC Buck-Boost Converter sterowany mikrokontrolerem ATtiny85. Projekt umożliwia konwersję napięcia zarówno w trybie obniżania (buck), jak i podwyższania (boost), z regulacją wyjścia przy pomocy sprzężenia zwrotnego.
